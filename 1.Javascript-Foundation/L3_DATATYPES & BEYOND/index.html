Mastering JavaScript Datatypes & Beyond

7 Main JavaScript Datatypes

  • 1. String: Represents text, e.g., "Hello, World!"
  • 2. Number: Represents both integers and floating-point numbers, e.g., 42, 3.14
  • 3. Boolean: Represents logical values, either true or false
  • 4. Null: Represents an intentional absence of value
  • 5. Undefined: Represents a variable that has been declared but not assigned a value
  • 6. Object: Represents a collection of properties, e.g., {name: "John", age: 30}
  • 7. Symbol: Represents a unique and immutable value, often used as object keys

Types Datatypes

  • Primitive Datatypes: If the value is a number , string , boolean , undefined , null , or symbol , it's a primitive value.
  • Reference Datatypes: (),{},[]
  • Example: let a = 12; //Primitive datatypes
    let a1 = 12.1; //Primitive
    let a2 = true; //Primitive
    let a3 = []; //Reference
    let a4 = false; //Primitive
    let a5 = null; //Primitive
    let a6 = function () {}; //reference

Truthy and Falsy Values

Falsy Values

  • 1. false
  • 2. 0 (the number zero)
  • 3. "" or '' (empty strings)
  • 4. null
  • 5. undefined
  • 6. NaN (Not-a-Number)
  • 7. document.all()

Truthy Values

  • 1. All values that are not falsy, including:
    • true
    • Non-zero numbers (e.g., 1, -42)
    • Non-empty strings (e.g., "Hello", 'World')
    • Objects (e.g., {})
    • Arrays (e.g., [])

Examples

  • Falsy Example: if (0) { console.log('Falsy'); } // No output
    if ('') { console.log('Falsy'); } // No output
  • Truthy Example: if (1) { console.log('Truthy'); } // Output: Truthy
    if ('Hello') { console.log('Truthy'); } // Output: Truthy
